CNC Label Automator (Python/QtPy)
An industrial automation tool designed to bridge the gap between CNC Optimization software and Thermal Label Printers. This application parses proprietary .tlf (Table Look-up Files) generated by CNC optimizers and translates that data into raw EPL/ZPL commands for instant, accurate production labeling.

🛠 The Problem
In manufacturing, CNC optimizers output complex job data that often requires expensive, third-party middleware to print correctly on thermal labels. Manual entry or generic drivers often lead to layout errors and production bottlenecks on the shop floor.

💡 The Solution
This application provides a streamlined, Python-based GUI that allows workshop operators to:

Select Job Folders: Quickly navigate to active CNC output directories.

Parse Proprietary Data: Intelligently extract part dimensions, material types, and job IDs from .tlf files.

ADD Default Details: Each part needs either Edging or Paint, by following simple naming conventions this data can be added to each label.

Generate Raw Commands: Bypass standard print drivers to send high-precision EPL (Eltron Programming Language) strings directly to the printer hardware.

🚀 Technical Features
QtPy GUI: A clean, functional interface for folder selection and job status monitoring.

Smart Parsing Engine: Regex-based extraction of data points from CNC-specific file formats.

Direct Socket Communication: Sends raw strings to printers via network or local ports, ensuring 1:1 layout fidelity.

Environment Managed: Utilizes Python virtual environments (venv) for stable, reproducible deployments in industrial settings.

💻 Tech Stack
Language: Python 3.x

GUI Framework: QtPy / PyQt

Data Handling: Native file I/O and String manipulation

Deployment: Batch-file automation for easy operator launch
